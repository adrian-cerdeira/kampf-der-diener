Das Spiel

Spielvorbereitung

Das Deck wird gemischt und verdeckt auf den Tisch gelegt. Die beiden Partnerkarten werden offen neben dem Deck hingelegt.
Beide Spieler ziehen eine Karte und legen sie offen auf den Tisch. Der Spieler mit der Karte, welche mehr Kristalle kostet, darf beginnen.
Die Karten werden wieder in die Decks gelegt und die Decks werden gemischt.
Jeder Spieler zieht fünf Karten und nimmt sie in die Hand, ohne sie den Gegenspieler zu zeigen.
Jeder Spieler beginnt das Spiel mit 0 Kristallen. Das Spiel kann beginnen.

Ein Spielzug

Bevor jeder Zug anfängt, wird deine Kristallleiste um eins erhöht. Im ersten Zug hast du ein Kristall, im zweiten Zug zwei und so weiter.
In deinem Zug kannst du deine Karten setzen und mit ihnen angreifen. Aber jede Karte muss eine Runde lang auf dem Feld liegen, bevor man sie für den Angriff verwenden kann.
Man sollte sich immer gut überlegen, wen man angreift. Wenn man nur andere Karten angreift, bekommt der Gegner nie Schaden, kann aber selbst nicht angreifen.
Wenn du nur den Gegner angreifst, bekommst du es früher oder später mit einer Masse voller gegnerischen Karten zu tun. Eine gute Mischung von beidem ist der richtige Weg zum Ziel.
Es müssen nicht alle Kristalle aufgebraucht werden und auch nicht alle Karten angreifen. Der Zug kann jederzeit beendet werden.
Mit jedem Zug erhält man mehr Kristalle, um seine Karten ins Spiel zu bringen.

Spielende

Wenn einer der beiden Spieler 0 Leben übrig hat, ist das Spiel beendet.
Derjenige mit 0 Leben hat verloren. Die Anzahl der Karten auf dem Feld spielen am Ende keine Rolle.
Bei vier Spielern müssen die Leben beider Spieler von einem Team auf 0 sinken.
